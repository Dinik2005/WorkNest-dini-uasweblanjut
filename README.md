langkah-langkah menjalankan project WorkNest
1. install composer
    composer install
   
3. membuat databases dengan ketik
   php artisan migrate
   
4. masukan data-data yang dibuat secara manual di code
   php artisan db:seed
   
5. setelah semua nya oke, lanjut dengan ketik
   php artisan serve
